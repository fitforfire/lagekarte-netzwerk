# Lagekarte-Netzwerk

Das Projekt [Lagekarte.info](https://www.lagekarte.info/) wird immer größer und bekommt auch eine immer größere Community.
Damit wir uns gegenseitig helfen können und auf die wichtigsten Wünsche eingehen können, haben wir diese Seite erstellt.
